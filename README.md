Weather Dashboard

A responsive Weather Dashboard Web App that allows users to check the current weather conditions for any city using the OpenWeather API.
It also provides a search bar for Google queries, dynamic backgrounds based on weather conditions, and a city weather history list.

🚀 Features

🌍 Search by City – Get real-time weather conditions for any city.

🌡 Weather Details – Displays temperature, condition, time, and icons.

🎨 Dynamic Background – Background changes based on current weather (Clear, Clouds, Rain, Snow, etc.).

🏙 City History – Keeps a list of previously searched cities with their weather.

🔍 Google Search – Integrated search bar to quickly Google any query.

ℹ About Section – Displays developer details and project purpose.

📱 Responsive Design – Works across devices with a modern UI.

📂 Project Structure
.
├── weather.html      # Main HTML file (UI for weather dashboard)
├── Styles.css        # Styles for layout, responsiveness, and themes
├── script.js         # Logic for fetching weather & updating UI

🔧 Setup & Installation

Clone the repository

git clone https://github.com/yourusername/weather-dashboard.git
cd weather-dashboard


Get an OpenWeather API Key

Go to OpenWeather

Create an account and generate an API key.

Replace the placeholder key in script.js:

const apiKey = 'YOUR_API_KEY';


Run the project

Simply open weather.html in your browser.

🖥️ Usage

Enter a city name in the search bar and click Search.

View real-time temperature, condition, and time.

The background adapts based on the weather.

Previous cities appear in the city list section.

Use the top search bar to Google anything directly.

Click the About button to view developer info.
